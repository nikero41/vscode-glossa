# Change Log

All notable changes to the "vscode-glossa" extension will be documented in this file.

## [0.1.0] - 2026-07-17

- Added syntax highlighting support for ΓΛΩΣΣΑ files with `.psg`, `.glo`, and `.gls` extensions.
- Added language configuration for ΓΛΩΣΣΑ files.

- Added unit tests setup
- Added CI workflow for linting, building, and testing.

- Updated prepublish script to run the full build.
