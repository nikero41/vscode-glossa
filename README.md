# Υποστήριξη ΓΛΩΣΣΑΣ

Επέκταση για το Visual Studio Code που προσθέτει υποστήριξη για τη γλώσσα ΓΛΩΣΣΑ.

## Δυνατότητες

- Χρωματισμός σύνταξης για αρχεία ΓΛΩΣΣΑΣ.
- Αναγνώριση αρχείων με κατάληξη `.psg`.

## Απαιτήσεις

Δεν υπάρχουν πρόσθετες απαιτήσεις.

## Ρυθμίσεις

Η επέκταση δεν προσθέτει ρυθμίσεις.

## Γνωστά Θέματα

Δεν υπάρχουν γνωστά θέματα.

## Σημειώσεις Έκδοσης

### 0.0.1

Αρχική έκδοση με βασική υποστήριξη χρωματισμού σύνταξης για τη ΓΛΩΣΣΑ.
