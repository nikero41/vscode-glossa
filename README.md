# Υποστήριξη ΓΛΩΣΣΑΣ

Επέκταση για το Visual Studio Code που προσθέτει υποστήριξη για τη γλώσσα ΓΛΩΣΣΑ.

## Δυνατότητες

- Χρωματισμός σύνταξης για αρχεία ΓΛΩΣΣΑΣ.
- Αναγνώριση αρχείων με κατάληξη `.glo`, `.gls` και `.psg`.

## Υποστηριζόμενες Καταλήξεις Αρχείων

Η επέκταση ενεργοποιείται για αρχεία με τις παρακάτω καταλήξεις:

- `.glo`
- `.gls`
- `.psg`

## Σχετικά

Η επέκταση αφορά τη γλώσσα ΓΛΩΣΣΑ και το περιβάλλον ΓΛΩΣΣΟΜΑΘΕΙΑ.

Περισσότερες πληροφορίες: https://glossomathia.gr
